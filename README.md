# Excel-VBA-Sales-Dashboard

## Description
Application de gestion commerciale développée sous Microsoft Excel et VBA, permettant de centraliser les données commerciales, d'automatiser le suivi des ventes, des commerciaux et des objectifs mensuels. La solution intègre une base de données, des formulaires de saisie, des calculs automatisés et un tableau de bord interactif pour suivre les KPI, comparer les performances aux objectifs et faciliter la prise de décision.

## Contenu du projet

- Tableau de bord interactif
- Base de données des ventes
- Suivi des objectifs mensuels par commercial
- Calcul automatique des indicateurs de performance
- Automatisation avec VBA

## Technologies

- Microsoft Excel
- VBA (Visual Basic for Applications)
- Formules Excel
- Tableaux Croisés Dynamiques
- Graphiques

## Objectif

Faciliter le suivi des performances commerciales et produire des indicateurs pour l'aide à la décision à partir des données de vente.
